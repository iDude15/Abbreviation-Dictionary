How to use: 

Type every abbreviation in uppercase letters, not in lowercase as it may not understand what are you entering.

For example, if you type NASA, it will expand the abbreviation for you and the result will be: National Aeronautics and Space Administration

If you want to view a list of available abbreviations, type "LIST".

If you want to view a list of upcoming abbreviations, type "UPCOMING_ENTRIES".

To exit the program, type "Q" and it will close the program.
